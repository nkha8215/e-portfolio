  • Gmail Clone

    Html, Css, JavaScript, ReactJS, Redux, Firebase

    This is a gmail clone I have built by implementing buttons and icons from googles Materiul UI. I also used Firebase Firestorm to fetch and store data of the emails that are composed. Furthermore Redux was implemented in combination with google authentication to determine wether the app is in a logged in or off state to give access to the user.

    When you log in with google you will be given access to the page and your icon will show in the top right. I recommend you to compose an email which will be added to the inbox list in front of you.

  • Twitter Clone

    Html, Css, JavaScript, ReactJS, Firebase

    This is a twitter clone I made by implementing icons from Googles Material UI. Furthermore, the tweets are fetched from and stored in firebase firestorm so that everyone who access the webpage has access to the same live feed. The project also implements Twitters Embed tools to display the widgets on the right.

    Feel free to post some tweets to add to the live feed :).

  • Google Clone

    Html, Css, JavaScript, ReactJS, Firebase

    This is a Google clone I made by implementing icons from Googles Material UI. Furthermore, the search results are derived from googles api, thus giving you the same results a normal google search would show. The Search results are clickable and will take you to the actual websites. This app is hosted on firebase hosting.
